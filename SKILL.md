# richman-skill Skill

Richman Skill 是一个面向专业投资人的选股工具，侧重：

- 从 **热门题材/赛道** 中出发（舆情+发帖热度）；
- 用 **K 线技术形态** 筛选买点（突破、底部反转等）；
- 自动生成结构化的 **推荐理由与风险提示**；
- 通过 OpenClaw 早晚报的形式，将结果推送给终端用户。

## 能力概述

1. **题材热度分析（计划中）**  
   - 抓取雪球 / 股吧 / 同花顺等公开网页数据，统计题材热度与增速。  
   - 输出当下最热的若干题材及其关键指标。

2. **技术形态识别（开发中）**  
   - 调用免费行情源获取日 K 数据（A 股起步，后续扩展港股/美股）。  
   - 根据 `patterns.yaml` 中可配置的规则识别突破、底部反转、首板等信号。

3. **推荐理由生成（计划中）**  
   - Richman Skill 不直接产生自然语言说明，而是准备结构化数据，供上层 LLM（如 Kimi / MiniMax）生成：  
     - 题材逻辑  
     - 公司/产业链位置  
     - 市场观点与研报摘要  
     - 风险提示。

4. **早晚报推送（计划中）**  
   - 早上 07:50：开盘前选股摘要。  
   - 晚上 21:30：收盘后复盘摘要。  
   - 通过 OpenClaw 的 `message` 工具发送到 Telegram / 飞书等渠道。

## 典型用法（设想）

在 Agent 的对话中，例如：

> “帮我跑一下 Richman Skill，看看今天 CPO 和高速互联这两条赛道里有没有新的技术信号标的，然后生成一份 Markdown 报告。”

上层 Agent 可以：

1. 调用 richman-skill 的 CLI 子命令或脚本，生成结构化 JSON/Markdown 报告。  
2. 对报告内容进行二次加工、翻译或摘要。  
3. 根据用户设置，决定是否在早 7:50 / 晚 21:30 自动推送报告。
