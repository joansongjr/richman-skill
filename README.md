# Richman Skill 📊

A股投研观点分析助手 — OpenClaw Skill

> 每天几十篇研报看不过来？粘贴进来，3 秒提取核心观点。

## 能做什么

- **观点提取**：上传研报 PDF / 纪要 DOCX / 截图 / 粘贴文本 → 自动提取结构化观点
- **信息分级**：四层可信度体系（L1公告 → L2研报 → L3纪要 → L4传闻），每条信息标注可信度
- **共识分析**：同一标的多方观点聚合，展示多空阵营对比 + 核心分歧点
- **投资简报**：一键生成带逻辑链的结构化简报

## 安装

```bash
clawhub install richman-skill
```

或者在 OpenClaw 对话中直接粘贴本仓库的 GitHub 链接，助手会自动安装。

## 使用

直接上传文件或粘贴文本即可，skill 会自动识别内容类型并处理。

### 快捷命令

| 命令 | 功能 |
|------|------|
| 直接上传/粘贴 | 自动提取观点 |
| `/batch` + `---` 分隔 | 批量提取多段内容 |
| `/consensus 标的名` | 查看多空共识 |
| `/brief` | 生成投资简报 |
| `/compare A vs B` | 对比两个标的 |
| `/list` | 列出已收集观点 |

### 输出示例

输入一段分析师观点：

```
中金电子首席今天观点：CPO 赛道明确看好，三个逻辑——
1. 英伟达 GB300 确认采用 CPO 方案
2. 国内光模块厂商已获得认证
3. 量产节奏超预期，Q3 开始放量
重点推荐中际旭创、新易盛
```

输出：

```
## 观点摘要

**来源**：中金公司 · 首席公开观点 · 今日
**分析师**：中金电子首席
**信息层级**：L2（机构半公开）

**标的**：中际旭创 / 新易盛
**方向**：看多

**核心观点**：
CPO赛道明确看好——
① GB300 确认采用 CPO 方案
② 国内厂商已获认证
③ Q3 开始放量

**置信度**：★★★★☆ L2·头部券商·首席·语气确定
```

## 覆盖方向

AI算力（光模块/CPO/AI芯片）· 半导体 · 新能源 · 消费电子 · 人形机器人 · 传媒/AI应用 · 军工

## 四层信息源体系

| 层级 | 名称 | 可信度 | 典型来源 |
|------|------|--------|---------|
| L1 | 官方公开 | ★★★★★ | 公告、财报、业绩会 |
| L2 | 机构半公开 | ★★★★☆ | 券商研报、晨会纪要 |
| L3 | 专业灰色 | ★★★☆☆ | 专家纪要、路演笔记 |
| L4 | 碎片即时 | ★★☆☆☆ | 群聊截图、市场传闻 |

每条观点都会标注信息层级和置信度评估，帮你区分"哪些信息值得信"。
